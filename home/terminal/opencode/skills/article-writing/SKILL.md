# Article Writing Skill

Article generation with research, planning, and review workflow.

## Workflow

```
scout -> planner -> reviewer
```

### scout
- Research topic
- Find relevant sources
- Gather context

### planner
- Outline structure
- Plan sections
- Define key points

### reviewer
- Review draft
- Check clarity
- Suggest improvements

## Usage

```bash
/article-writing "Topic: machine learning trends"
```

## Output

Generated article with:
- Title
- Introduction
- Body sections
- Conclusion
- References
